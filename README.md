# node-grpc-example
Simple Node application using [gRPC](https://grpc.io/)

# Requirements
   - node 12+
   - npm

## Running the project

### Install dependencies
```shell script
npm install
```
### Start server
```shell script
npm start
```
### Insert Person
```shell script
npm run insert-person
```
### List all people
```shell script
npm run list-person
```

## Running server with docker compose
```shell script
docker-compose up
```
